# 🚀 iCodeGuru Web Development BootCamp 2026

Welcome to the official repository for the **iCodeGuru Web Development BootCamp**! This repository contains the source code, project files, and resources developed during our intensive hands-on training sessions.

Our mission is to take aspiring developers from the fundamentals of the web to building production-ready, full-stack applications.
