## Week 1: Web Foundations & Introduction to HTML
### Goal:
Understand how the web works and build a simple, structured webpage using semantic HTML.

### Core Topics & Keywords:
   -   How the Web Works: Client-Server Model, HTTP/HTTPS, DNS, Web Browsers.
   -  HTML Syntax: Elements, Tags, Attributes, Nesting, Comments.
   -   Document Structure: `<!DOCTYPE html>`, `<html>`, `<head>` (with `<meta charset>`, `<title>`), `<body>`.
   -   Text & Content: Headings (`<h1>-<h6>`), Paragraphs (`<p>`), Bold (`<strong>`) & Italic (`<em>`), Lists (`<ul>`, `<ol>`, `<li>`), Line Breaks (`<br>`), Horizontal Rule (`<hr>`).
   -   Links & Media: Anchor Tag (`<a href="">` with `target="_blank"`), Images (`<img src="" alt="">`), File Paths (Absolute vs. Relative).
   -   Semantic HTML5: `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<footer>`.

### Project:
Create a simple, multi-page personal profile website (e.g., "About Me," "My Hobbies," "Contact") using only semantic HTML.

## Week 2: Styling with CSS
### Goal:
Learn to control the visual presentation of HTML content, creating aesthetically pleasing and responsive layouts.

### Core Topics & Keywords:
* **CSS Basics:** Syntax, Selectors (Element, Class `.`, ID `#`), The Cascade, Specificity.
* **Including CSS:** Inline Styles, `<style>` Tag, External Stylesheets (`<link rel="stylesheet">`).
* **The Box Model:** `width`/`height`, `padding`, `border` (`border-radius`), `margin`, `box-sizing: border-box`.
* **Essential Styling:** `color`, `background-color`/`background-image`, `font-family`, `font-size`, `font-weight`, `text-align`, `line-height`, `text-decoration`.
* **Layout - Flexbox:** `display: flex`, `flex-direction`, `justify-content`, `align-items`, `flex-wrap`, `gap`.
* **Responsive Design:** Relative Units (`%`, `rem`, `vw`/`vh`), `max-width`, Media Queries (`@media (max-width: ...)`).
    

### Project:
Style the Week 1 HTML project. Implement a responsive navigation bar using Flexbox and ensure the layout looks good on both desktop and mobile.
